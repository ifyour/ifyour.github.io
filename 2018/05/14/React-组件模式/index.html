有状态组件 vs 无状态组件

正如 WEB 服务有静态和动态之分，React 组件也有有状态和无状态的区分。

  • 有状态组件：在应用中组件可以拥有自身状态并操纵它；
  • 无状态组件：只接收属性进行效果呈现。
