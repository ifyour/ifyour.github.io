工具及文档

May 6, 2018

Interesting tools

  • Rico’s cheatsheets - 记忆小抄, 长话短说各种开发文档
  • Carbon - 带样式分享代码
  • CodePen - 在线测试前端代码
  • CodeSandbox - 代码沙盒，可以分享带有依赖的前端项目
  • Prettier - 在线格式化 JavaScript 代码
  • Text-formatter - 格式化文本, 中英文之间留空格以及英文标点
  • DevDocs - 一站式开发文档集合
  • Grammarly - 英文语法错误检查
  • JSONPlaceholder - RESTful API 假数据模拟
  • JSON2YAML - JSON 和 YAML 互转
  • JSON Parser - JSON 格式化
  • RegExr - 在线正则测试及调试
  • Regulex - 正则表达可视化, 快速理解正则表达式
  • CSSMatic - 生成镜像渐变, 盒子阴影等
  • CSS Gradient - 专业生成镜像渐变、线性渐变
  • SassMeister - Sass 演练场, 可以输出 AutoPrefixer 的结果
  • KeyframesApp - CSS3 动画关键帧生成工具
  • CSS Sprites Generator - CSS 精灵生成工具
  • JSNice - 反编译已压缩的 JS 代码, 理解变量含义
  • Love2.io - 开源技术文档分享平台
  • WakaTime - 记录每天的编码情况, 使用什么语言及统计时长
  • Best of JavaScript - 最流行的前端开源项目清单
  • 蓝湖 - 高效的产品设计协作平台, 标注切图很好用
